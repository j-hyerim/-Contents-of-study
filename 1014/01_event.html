01_jquery_event

javascript 언어로 버튼을 클릭하면 '클릭이벤트 입니다.' 라는 메세지를 띄우기

        1)
        onclick="alert('click event');"

        2)
        onclick="but()"

        function but() {
            alert('클릭이벤트입니다.');
        }
    

        3)
        id="btn"

        let btn = document.getElementById('btn');
        btn.addEventListener('click',function(){
            alert('click event');
        });

웹 문서에서 버튼을 클릭하거나 마우스를 움직이는 모든 행위를 '이벤트'라고 한다. 그리고 이벤트 발생 시 함수의 실행문이 수행되도록 이벤트를 지정하는 것을 '이벤트 핸들러' 라고 한다.

이벤트의 종류

  • 마우스 이벤트
  • 키보드 이벤트
  • 포커스 이벤트
  • 폼양식 이벤트
  • 윈도우 이벤트 - 화면 스크롤(scroll)

이벤트 작성방법

  1. 단독 이벤트 - $('이벤트 대상').이벤트 메서드(function(){실행문;});
  2. $('이벤트 대상').on('이벤트 종류1,이벤트종류2...',function(){실행문;});

마우스 이벤트와 양식 이벤트 사용하기

  1. click - 마우스로 대상을 클릭하엿을 때 이벤트 내용이 실행
  2. dblclick - 마우스로 대상을 더블클릭 하였을 때 이벤트 내용이 실행
  3. mouseenter - 마우스로 대상을 오버하였을 때 이벤트 내용이 실행(=mouseover)
  4. mouseleave - 마우스로 대상을 오버하였다가 떠났을 때 이벤트 내용이 실행(= mouseout)
  5. hover - 마우스로 대상을 오버하였을 때, 뺏을 때 둘다 각각의 이벤트 내용이 실행
  6. focus - 마우스로 클릭하여 포커스가 발생하였을 때 이벤트 발생
  7. blur - 폼 양식에서 포커스가 해제되었을 때 이벤트 발생
  8. toggle - 대상 요소를 클릭할 때마다 지정된 함수를 호출하여 실행
  9. each - 개별 처리
click event
dblclick event
mouseenter event
mouseleave event
hover event
toggle event

위 toggle 박스를 클릭하면 현재 문단 태그 요소에 서식이 변경됩니다.
toggle이벤트는 제이쿼리 1.8 버전 이하에서만 동작됩니다.
