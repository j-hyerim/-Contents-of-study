1. javacript는 html문서안에서 삽입되어 작동된다.
  2. 기본적으로 css 언어와 같이 head 안에서 작성되며, 필요에 따라 body tag 안에서도 작성이 가능하다.
  3. javacript는 주석을 사용할 때 2가지 종류가 있으며, 한줄주석은 '//', 2줄 이상의 주석은 '/* */'기호를 사용한다.
  4. html, css tag element는 대소문자를 구별하지 않지만, javacript는 대,소문자를 구별해야 한다.
  5. javacript는 문자데이터를 입력시 ''홑따옴표 기호, ""쌍따옴표 기호를 사용하여 묶어준다.
  6. javacript는 css처럼 문서내에서 작성하는 것을 내부 스크립트라고 하며, 외부문서로 별도로 저장하여 html문서에 삽입하여 연결하는 것을 외부스크립트(파일명.js)라고 한다.
  7. javacript의 선언은 css처럼 script, /script 로 선언하고 종료한다.
