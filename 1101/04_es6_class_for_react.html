04_es6_class_for_react

  • java에서는 ㄴ클래스를 객체로 만들기 위한 설계도라고 부릅니다.
  • 객체를 생성하기 위한 필드와 메소드가 정의되어 있는 설계도 입니다.
  • 예시로 동일한 성계도를 이용해 여러 대의 자동차를 생산하는 것과 같은 의미입니다.
  • 클래스는 일종의 함수지만 function 키워드 대신 class 키워드를 사용합니다.
  • 속성은 constructor() 메소드 내부에 할당됩니다.
  • class는 사실 함수이기 때문에 함수선억식과 변수선언식 둘다 선언 가능합니다.

1. 함수 선언식의 class 문법

    * 문법 :
    class 클래스 명{

        this.속성명 = 값;

    }

      class Car{
        constructor(name){
          this.brand=name;

      }

2. 부모클래스를 상속받는 자손 클래스

  • 상속 : 부모클래스의 멤버(필드, 메소드)를 자식 클래스에게 물려주는 것입니다.
  • 부모클래스는 보통 상위 클래스, 슈퍼 클래스라 부릅니다.
  • 자손 클래스는 보통 하위 클래스, 파생 클래스라 부릅니다.
  • 이렇게 나눠서 쓰는 이유는 이미 잘 개발된 클래스를 재사용해서 새로운 클래스를 만들기 때문에 중복되는 코드를 줄여주기 때문입니다.
  • 부모 클래스의 수정으로 무든 자식 클래스가 수정되는 효과를 가져와 유지보수시간을 최소화 시켜줍니다.
