제이쿼리 문서 객체 조작을 활용한 이미지 갤러리( 모달, 내비게이션, 캡션 )

구현순서
목록태그로 내비게이션 메뉴 5개를 만들고 클릭 시 해당 이미지 목록만 보이게 한다.
콘텐츠 영역에 목록 태그를 사용하여 12개의 li 태그에 각각 마우스 오버시 캡션 내용이 나타나게 하낟.
이미지 목록에 마우스 클릭 시 해당 사진의 src 속성 값을 변수에 담아 모달 윈도를 띄운다.
모달 윈도에서 '닫기'를 클릭 시 해당 이미지가 사라지게 한다.
갤러리 내비게이션을 클릭시 해당 메뉴에만 act 서식을 적용하도록 한다.
