01jquery_start

jQuery? javascript의 생산성을 향상시켜주는 javascript의 라이브러리 언어이다.
javascript의 라이브러리란? 자주 사용하는 코드들을 재사용할 수 있는 형태로 가공하여 프로그래밍의 효율성을 높여주도록 만든 코드들의 집합이다.

1. jQuery의 특징

  • element(html tag element)를 선택하는 강력한 기능을 제공한다. (css selector < jquery selector)
  • 선택한 요소를 제어할 수 있는 다양한 함수를 제공한다.
  • javascript의 자주 사용하는 기능을 라이브러리화한 언어이다.

2. jQuery 라이브러리 연결 방법

  • jquery 라이브러리 파일을 직접 연결하기 - jquery 공식사이트에서 라이브러리 파일을 다운로드하여 html 문서에 head 요소 안에 직접 연결하여 사용한다.
  • CDN (Content Delivery Network) 주소로 연결하기 - 서비스를 제공하는 업체로부터 링크 주소를 사용해서 문서에 열결하여 사용한다. `제이쿼리 문법(작성법)`

3. jQuery 문법(작성법)

  • $('대상').메소드();
  • jquery('대상').메소드();

4. jQuery '$' 역할

jquery에서는 '$'기호를 맨앞에 작성한다. 이것은 괄호()안에 넣을 대상을 선택하기 위함이다.
$(document),$('.txt01'), $('body'), $('window')...

5. jQuery '$' 다음으로 들어가는 것은?

  • html tag element
  • css selector
  • javascript 객체 (document, window...)

6. jQuery 기본 wrapper(선언하기)

  • $(document).ready(function(){실행할 내용;})
  • $(function(){실행할 내용;})
  • jQuery(function(){실행할 내용;})
  • $(function(){실행할 내용;})(jQuery)
