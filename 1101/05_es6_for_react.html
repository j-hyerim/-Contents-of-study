05_es6_for_react(비구조화 할당)

-비구조화 할당이란 배열이나 객체의 속성 혹은 값을 해체하여 그 값을 변수에 각각 담아 사용하는 자바스크립트 표현식입니다.

1. 기존 배열 값을 해체(이전 방법)

    var yoil = {'일','월','화','수','목','금','토'};]
    document.write(yoil);
    document.write(yoil[0]);

2. 비구조화 할당 방식으로 배열 선언과 호출하기

    //const [변수명1,변수명2,변수명3,변수명4,변수명5] = [값1,값2,값3,값4,값5];
    const [cat, dog, animal] = ['고양이','강아지','팬더'];
    console.log(cat);
    console.log(dog);
    console.log(animal);
