01_javascript_string

01_javascript_string의 내장객체 중에서 활용도가 많은 객체이며, 문자열 값을 인덱스 번호를 통해 값을 찾거나 다른 문자로 변경이 가능하다.

아이디, 패스워드 입력창 문자열 개수 체크, 단문 메세지 문자열 개수 체크 패스워드 문자열 찾아서 다른 문자열로 변경, 이미지 갤러리 등...

문자열 객체는 다른 내장 객체와는 다르게 new키워드를 생각해도 된다.

mathod의 종류
replace - 문자치환
split - 문자열 분리
substring - 인덱스번호 구간까지 문자데이터를 찾아준다.
length - 문자열 개수를 체크해준다.
indexof - 문자열과 일치하는 인덱스 번호를 반환한다.
