02_jquery_effect

effect란? 사용자가 지정한 html 요소를 선택하여 숨기거나 나타내기, 페이드 효과, 슬라이드 다운, 업, 애니메이션 기능을 사용하기 위한 메소드들이 있다.

  • show(); - 선택한 요소를 보이게 한다.
  • hide(); - 선택한 요소를 숨긴다.
  • toggle(); - 선택한 요소를 숨기거나/나타낸다.
  • slideDown(); - 선택한 요소를 아래로 펼친다.
  • slideUp(); - 선택한 요소를 위로 접는다.
  • slideToggle(); - 선택한 요소를 아래로 펼치거나/ 위로 접는다.
  • fadeIn(); - 선택한 요소를 서서히 나타낸다.
  • fadeOut(); - 선택한 요소를 서서히 사라지게 한다.
  • fadeTpggle(); - 선택한 요소를 나타내거나/ 사라지게 한다.
  • fadeTo(); - 선택한 요소를 지정한 시간만큼 서서히 사라지게 한다.

show, hide 효과는 전체메뉴 숨기기/나타내기, 모달팝업 숨기기/나타내기

fade 효과는 메인 이미지 서서히 나타내기/ 서서히 사라지기

slide 효과는 내비게이션 작성 시 2단 메뉴에 적용
