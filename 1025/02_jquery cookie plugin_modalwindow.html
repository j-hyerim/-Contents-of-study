02 02_jquery cookie plugin_modalwindow

jquery cookie plugin address : https://plugins.jquery.com/cookie/

coocke : 서버에서 클라이언트에게 특정 정보를 저장하는 데이터이다.

사용자 이름, 아이디, 바움ㄴ 획수 등을 기록하여 차후 저 접속 시 활용한다.

제이쿼리 쿠키 작성법

$.cookie('쿠키이름','쿠키값',{expires:만료일, path:'저장경로});

모달윈도 = 팝업 레이어

Modal Window 는 사용자 인터페이스 디자인 개념에서 자식 윈도부터 부모 윈도우로 돌아가기 전에 사용자의 상호동작을 요구하는 창을 말한다.

Modal window 는 부모창을 배경으로 어둡게 하고 자식 요소가 화면의 가운데 배치하도록 하며, 오른쪽 상단에 '닫기' 버튼을 만들어서 닫을 수 있도록 한다.
