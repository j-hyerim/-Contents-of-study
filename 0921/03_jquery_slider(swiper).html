03_jquery_slider(swiper)

http://swiperjs.com 스와이퍼 슬라이드 플러그인 - 모바일 터치 동작에 최적화 되어 있으며 여러가지 형태의 슬라이드를 제작할 수 있다.

작성방법
1. css 폴더 안에 swiper.css 문서를 저장하고 html 문서에 link를 삽입한다.
2. script 폴더 안에 swiper.js 문서를 저장하고 html 문서에 script를 삽입한다.
3. 슬라이드를 삽입하기 위한 영역에 div class="swiper-container" tag로 영역을 만든다.
4. swiper-container 이름 옆에 class 명을 하나 더 추가한다. 슬라이드가 여러개일 경우 겹치지 않게 하기 위함이다.
5. container 안에 .swiper-wrapper 영역을 만든다.
6. swiper-wrapper 영역안에 들어갈 이미지 영역을 이미지 개수와 같이 .swiper-slide로 만든다.
7. 좌 우 방향버튼을 삽입할땐 div class="swiper-button-next" / div class="swiper-button-prev" 코드를 적용한다.
8. 컨트롤 버튼 삽입 시 div class="swiper-paginatio" 코드를 적용한다.
9. body 아래 마지막 줄에 script 소스코드를 넣어서 슬라이드가 움직이게 한다. (class명 확인 필요)
