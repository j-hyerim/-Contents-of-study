제어문이란? 프로그램의 실행을 제어하기 위해 사용하는 실행문으로서 조건문(if, else, else if), 선택문(switch case), 반복문(while, do while, for)이 있다.

1. 조건문 - if, else, else if

  • if - 조건식이 만족할 때만 실행되는 조건문이다.
  • else - 조건식이 만족하면 true, 만족하지 않으면 false가 나오도록 하는 조건문이다.
  • else if - 조건식이 여러개일 경우 다양한 결과 값을 얻기 위한 조건문이다.
