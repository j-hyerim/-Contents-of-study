03_javascript_data

Data 객체는 현재 또는 특정 날짜와 시간에 대한 정보를 제공하는 객체이다.

[ 작성법 ] var 변수명 = new Date();

날짜와 시간정보를 제공하는 속성

  • getFullYear() : 날짜정보에서 연도에 대한 정보를 제공
  • getMonth() : 날짜정보에서 월(0-11)에 대한 정보를 제공
  • getDate() : 날짜정보에서 일에 대한 정보를 제공
  • getDay() : 날짜정보에서 요일에 대한 정보를 제공
  • getHours() : 날짜정보에서 시간(0-23)에 대한 정보를 제공
  • getMinutes() : 날짜정보에서 분에 대한 정보를 제공
  • getSeconds() : 날짜정보에서 초(0-59)에 대한 정보를 제공
  • getMilliseconds() : 날짜정보에서 밀리초 (1/1000)에 대한 정보를 제공
