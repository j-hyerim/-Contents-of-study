04_javascript_control_for

반복문은 작업을 반복수행 할 때 사용하며, 조건문이 프로그램을 지적으로 만든다면 반복문은 프로그램을 강력하게 만드는 역할을 한다.

주의할 점 - 조건을 잘못 입력하면 무한 루프에 빠지는 오류가 생기며, 브라우저에 화면 멈춤 현상이 발생된다.

  • while - 반복조건
  • do while - 반복조건
  • for - 반복횟수

for문 작성 방법

        for(초기값; 조건식; 증감식){
            실행할 내용;
        }
