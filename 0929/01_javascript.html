01_javascript_객체(window)

browser 객체는 사용자가 사용하는 웹 브라우저의 창에서 다양한 동작이나 속성을 설정하고자 할 때 사용하는 객체이다.

window 객체

window 객체는 browser 객체 중 최상위 객체로서 현재 실행되고 있는 브라우저 창을 나타낸다.

window 객체의 메소드 종류
alert() : 경고창 메세지를 띄울 떄 사용한다.
prompt() : 데이터를 입력받을 수 있는 명령 창이다.
confirm() : 질문 내용에 대한 확인, 취소를 하고자 할 때 사용한다.
setInterval() : 일정 시간동안 작성한 코드나 함수를 실행하고자 할 떄 사용한다.
clearInterval() : 일정시간동안 설정한 함수나 코드에 적용된 시간을 제거할 때 사용한다.
open() : url 주소를 삽입하여 현재 페이지가 아닌 새로운 창에서 페이지가 나오도록 한다.
close() : 윈도우 창을 닫을 때 사용한다.
