1. find()메서드

find():함수의 조건을 통과한 첫번째 배열 요소값을 반환하는 메서드입니다.

    find(함수명)
    var nums = [4,9,16,25,30];
    function test(x){
      return x >20;
    }
    document.write(nums.find(test));
